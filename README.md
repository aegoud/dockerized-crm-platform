# Dockerized CRM Platform

## Overview

This project demonstrates a full-stack CRM (Customer Relationship Management) application deployed using Docker and Docker Compose.

The application consists of:

* React Frontend
* Node.js Backend API
* MongoDB Database

## Architecture

Frontend (React)
↓
Backend (Node.js)
↓
MongoDB

## Technologies Used

* Docker
* Docker Compose
* Kubernetes
* React
* Node.js
* MongoDB
* Git
* Linux

## Features

* Multi-container deployment
* Container networking
* MongoDB integration
* Docker Compose orchestration
* Kubernetes deployment manifests
* Service communication between containers
* Mobile and desktop accessibility

## How to Run

### Build Images

docker compose build

### Start Application

docker compose up -d

### Access Application

Frontend:
http://localhost:3000

Mobile:
http://192.168.1.110:3000

Backend:
http://localhost:5000
